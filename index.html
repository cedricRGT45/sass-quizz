Question 5 : Illustration par du code

Vous avez le code CSS suivant :


.nav {
  text-align: right;
}
.nav .nav__link {
  font-size: 15px;
  padding-left: 30px;
}
.nav .nav__link .nav__link--purple {
  color: #a5b4fc;
}

Lequel de ces exemples de code représente la même structure en Sass (SCSS) avec un code plus DRY ?

A.


.nav {
  text-align: right;
  .nav__link {
    font-size: 15px;
    padding-left: 30px;
    .nav__link--purple {
      color: #a5b4fc;
    }
  }
}

B.


.nav {
  text-align: right;
}

.nav__link {
  font-size: 15px;
  padding-left: 30px;
}

.nav__link--purple {
  color: #a5b4fc;
}
  • A.
  • B.
